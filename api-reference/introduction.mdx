## Welkom bij EPAdossier API

De EPAdossier API stelt je in staat om programmatisch te integreren met ons platform. Gebruik onze REST API om leads om te zetten in projecten, dossiers aan te maken en meer.

<Note type="info">
  **API Sleutel Vereist**: Alle API endpoints vereisen authenticatie via een
  API-sleutel in de `X-API-Key` header.
</Note>

## Authenticatie

Alle API endpoints worden geauthenticeerd met behulp van API-sleutels. Je kunt je API-sleutel vinden in je EPAdossier account onder **Instellingen** â†’ **API Sleutels**.

```bash
# Voorbeeld header
X-API-Key: epa_jouw_secret_key
```

## Base URL

```
https://app.epadossier.nl/api/v1
```

## Rate Limiting

- **100 requests per minuut** per API-sleutel
- **1000 requests per uur** per API-sleutel
